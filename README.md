## Как запустить проет ?

В первую очередь необходимо запустить (backend) hackathon-3-back
Запустить терминал и набрать следующие команды:

```bash
npm i
npm run start
```

Затем необходимо запустить (frontend) hackathon-3-front
Запустить терминал и набрать следующие команды:

```bash
npm i
npm run start
```
